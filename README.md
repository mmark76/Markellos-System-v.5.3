## **3. Core Functionality**

### **3.1 Input Handling**

* Load complete PGN games
* OR type moves manually in SAN notation
* Automatic parsing through `pgn.js`

### **3.2 Mnemonic Generation**

Each move maps to:

* **Temporal mnemonic** (sequence logic)
* **Spatial mnemonic** (board structure / loci)
* **Character mapping** (piece personalities)
* **PAO encoding**
* **Verse/poetic associations**
* **Foundational patterns**

Generated by the libraries inside `/js/libraries/`.

### **3.3 Mnemonic History Panel**

* Shows move-by-move stories
* Allows **manual overrides** saved locally
* Updates dynamically when loading new games

### **3.4 User Custom Library**

`user_custom.js` allows users to override the built-in libraries with completely personalized mappings.
